# use this project
* 在config.json配置文件里根据json对应格式配置好账号密码后(这里多个账户我使用了一个密码,有需求提issue)
* go build
* 执行exe文件
# 2021/4/9下午4:39:48 更新：
* 增加命令行参数 动态读取只能账号密码

* **go build** 之后运行exe文件：

* 声明**-m**参数 0为默认开以不用添加,直接回车执行

* `hlx.exe`

* 1：单账户签到

* `hlx.exe -u xx.qq.com -p 密码写这里`

* 2: 多账号签到 以逗号分割

* ```go
  hlx.exe -u xx.qq.com,xx.qq.com -p 123,123
  ```
