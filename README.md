* **cd siginin**

* **go build** 之后运行exe文件：

* 声明 **-m** 参数 0为默认 可以不用添加 , 直接回车执行

* `hlx.exe`

* 1：单账户签到

* `hlx.exe -u xx.qq.com -p 密码写这里 -m 1`

* 2: 多账号签到 以逗号分割

* ```go
  hlx.exe -u xx.qq.com,xx.qq.com -p 123456,654321 -m 2
  ```
